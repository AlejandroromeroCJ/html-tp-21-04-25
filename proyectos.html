Sistema de Control de Stock

Aplicación web desarrollada con HTML, CSS y JavaScript que permite gestionar el stock de productos de manera sencilla e intuitiva. Incluye funcionalidades para agregar, editar y eliminar productos, así como visualizar el total de unidades y calcular el valor del inventario. La interfaz es dinámica y responsive, y los datos se almacenan localmente en el navegador mediante. Este proyecto fue una práctica fundamental para fortalecer mis conocimientos, lógica de programación y diseño de interfaces.
